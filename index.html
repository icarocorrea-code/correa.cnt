🔥 Firebase — Configuração do banco de dados

📋 Passo a passo para ativar o Firebase 1. Acesse console.firebase.google.com e crie um projeto gratuito.
2. Vá em Autenticação → E-mail/senha → Ativar e crie seu usuário.
3. Vá em Firestore Database → Criar banco (modo produção ou teste).
4. Em Configurações do projeto → Seus apps → Web, copie o firebaseConfig.
5. Cole as credenciais no bloco firebaseConfig no topo do arquivo HTML.
6. Pronto! Leads e clientes passam a ser salvos em tempo real. ✅
As credenciais são inseridas diretamente no arquivo HTML (bloco firebaseConfig).

WhatsApp — Z-API
Como obter as credenciais Z-API 1. Acesse z-api.io e crie uma conta (plano gratuito disponível).
2. Crie uma instância e escaneie o QR Code com o WhatsApp.
3. Copie o Instance ID e o Token abaixo.
4. Configure o Webhook para https://SEU_DOMINIO/webhook/whatsapp.
Meta — Instagram / Facebook / Ads
Como obter o Access Token Meta 1. Acesse developers.facebook.com → Meus Apps → Criar App.
2. Adicione os produtos: Messenger, Instagram Graph API, Marketing API.
3. Em Ferramentas → Explorador de API do Graph, gere um token de longa duração.
4. Permissões necessárias: pages_messaging, instagram_manage_messages, ads_read.
Google Ads API
Como obter as credenciais Google Ads 1. Acesse Google Ads API e solicite um Developer Token.
2. No Google Cloud Console, crie um projeto e ative a Google Ads API.
3. Crie credenciais OAuth2 (ID do cliente + Secret).
4. Use o OAuth Playground para gerar seu Refresh Token.
